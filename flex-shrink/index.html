flex-shrink デモ

コンテナ幅: 400px / 各アイテムの flex-basis: 200px（合計 600px → 200px 分のオーバーフロー）

1. 全て flex-shrink: 1（デフォルト）

A (1)
B (1)
C (1)

→ オーバーフロー分を均等に縮小して収める

2. 全て flex-shrink: 0（縮まない）

A (0)
B (0)
C (0)

→ 縮小しないためコンテナからはみ出す（横スクロール可）

3. flex-shrink を 1 : 3 : 1 に設定

A (1)
B (3)
C (1)

→ B がオーバーフロー分の 3/5 を吸収し、より多く縮む

4. B だけ flex-shrink: 0（A, C は 1）

A (1)
B (0)
C (1)

→ B は flex-basis: 200px を維持し、A と C だけが縮む
