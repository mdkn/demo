flex-basis デモ

全例で flex-shrink: 0 を設定（特記がない限り縮小しない）

1. flex-basis: auto（デフォルト）

Short
Medium text
Longer content here

→ 各アイテムのコンテンツ幅がそのまま反映される

2. flex-basis を固定値で指定（100px / 200px / 150px）

A (100px)
B (200px)
C (150px)

→ コンテンツに関係なく、指定した幅になる

3. flex-basis: 0 + flex-grow: 1

Short
Medium text
Longer content here

→ ベースが 0 なので、コンテンツ量に関係なく 完全に均等幅 になる

4. flex-basis: auto + flex-grow: 1（3 との比較）

Short
Medium text
Longer content here

→ ベースがコンテンツ幅なので、余白は均等でも 最終幅は不均等 になる

5. flex-basis を % で指定（20% / 50% / 20%）

A (20%)
B (50%)
C (20%)

→ コンテナ幅に対する割合でベースサイズが決まる（gap 分で合計 100% にはならない）
